# Извлечение JavaScript из email-body.html

## Цель

Создать отдельный JavaScript-файл с читаемым эквивалентом встроенного кода из `CVE-2026-42897/email-body.html`.

## Объём

- Перенести inline-обработчики `onerror` в обычный JavaScript.
- Собрать фрагменты JavaScript, разрезанные по атрибутам `class`.
- Декодировать Base64-фрагменты в читаемый код.
- Сохранить порядок выполнения и поведение исходного кода.
- Не менять `email-body.html`, `outlook.html` или `sendMessage.ps1`.

## Результат

Новый файл `CVE-2026-42897/email-body.js` с самостоятельным читаемым кодом, который можно подключить к HTML в дальнейшем.

## Проверка

Проверить, что все JavaScript-фрагменты из HTML представлены в новом файле, порядок операций сохранён, а в рабочем дереве изменён только новый JS-файл и эта спецификация.
